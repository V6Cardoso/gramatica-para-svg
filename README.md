# gramatica-para-svg
EC6 Compiladores Desafio 4


Exemplo
Para uma entrada:

n : 6

ω : F-f-f

δ : 120

p1 : F -> F-f+F+f-F

p2 : f -> ff

Onde f minúsculo representa desenhar para frente e um ângulo inicial de -60 graus. Esse será o resultado:
